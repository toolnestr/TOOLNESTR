# Low-Competition, Higher-RPM Tool Ideas

Strategy: **Specific beats generic. Boring beats popular. New beats established.**
Don't build "Calculator." Build "Netherlands net salary calculator 2026."

Keep all existing generic calculators as filler + entry doors, but build every
NEW tool from the list below. Each row = one tool = one page = one chance to rank.

Legend:
- **RPM** = how well ads/affiliates pay ($ = low, $$$$$ = high)
- **Comp** = competition (Low = easier to rank #1)

## How to use this list

1. **Pick the Low-competition rows first** — fastest wins.
2. **Lead with Finance + Country-specific** — best earnings per visit.
3. **Grab a few AI tools now** — the ranking window is open and closing.
4. For each new tool, add **"Related tools" links** back to your existing
   calculators (and vice-versa) so visitors view more pages.
5. Track which ones get impressions in **Google Search Console**, then build
   more tools in whatever category starts working.

## Reminder: traffic value, not traffic volume
- 30k visits in a $1 RPM niche  ≈ $30/month
- 30k visits in a $15 RPM niche ≈ $450/month
Niche choice matters ~10× more than raw visitor count. Build for value.
